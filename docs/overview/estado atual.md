# P2App - Estado Atual

## Stack atual
- FastAPI
- PostgreSQL
- SQLAlchemy
- Alembic
- JWT para autenticação
- Passlib/Bcrypt para hash de senha

## Módulos já existentes
- Usuários
- Autenticação
- Chamados

## Funcionalidades já prontas
- Cadastro público de usuário cliente
- Listagem de usuários restrita a admin
- Login com geração de access token
- Endpoint /auth/me
- Criação de chamado por cliente autenticado
- Listagem de chamados:
  - admin vê todos
  - cliente vê apenas os próprios

## Situação real do projeto
O projeto já saiu da fase de CRUD básico de usuário.
O domínio inicial de chamados já foi iniciado.

## Ponto atual
Estamos na fase de consolidação do módulo de chamados e alinhamento entre código e banco.
